**AI SEO Blog Pack**

**Project Title**



AI SEO Blog \& Content Cluster Generator



**Business**

Glow \& Grace Salon



**Location**

Hyderabad



**Objective**



The goal of this project is to build a prompt system that generates SEO-optimized blog posts and content clusters for business websites using AI tools such as ChatGPT, Claude, or Gemini.



**Features**



Generate SEO blog outlines (H1–H3)

Write long-form SEO blogs

Create content clusters

Optimize blogs for local SEO

Generate FAQs and meta descriptions

Improve internal linking strategies
